# Reinforcement Learning code in LANADA

## requirements
Tested with python3.6\ 
```
pip install -r requirements.txt
```
gym==0.10.5\
matplotlib==2.2.4\
tensorflow==1.14.0


After installing above, Install gym-maze.
```
git clone https://github.com/lanada/gym-maze.git
cd gym-maze
python setup.py install
```
